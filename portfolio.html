Alguns dos projetos criados pela nossa equipe:

1. Q-Manager — Sistema de Gestão de Tarefas
Um sistema criado para organizar tarefas, definir prioridades e acompanhar o progresso de equipes.
2. FinTrack — Aplicativo de controle financeiro pessoal
Aplicativo voltado para o controle financeiro, ajudando usuários a gerenciar despesas e metas.
3. EduSys — Plataforma de gestão escolar
Sistema para escolas e professores gerenciarem alunos, turmas e notas de forma integrada.
4. API Connect — Serviço REST para autenticação e login seguro
API desenvolvida para integração de login seguro em múltiplas plataformas.
